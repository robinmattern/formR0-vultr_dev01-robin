Shouldn’t every computer professional understand the total picture? You can’t go wrong with more knowledge about all of the disciplines.

The goal of the formR tutorial is to give you a step by step guide to building a complete environment for a secure server on the cloud serving web pages that access and manipulate data via the Internet.

2. Setup Remote Server: Create Vultr Ubuntu - Harden Ubuntu - Install: Web Server, App Server, and Data Server - Run SSL Website

3. Customize FRApps: My HTML App - My Javascript App - My React App

4. Run formR: Deploy My Apps - Clone formR - Deploy formR - World Database - Authentication - User Administration - World Applicatication

5. Customize formR: Use Your Own Data - Deploy Your Own App

6. Markdown Reference - Introduction - Markdown Examples - CSS Variables - Docsify Templates - FormR Example
